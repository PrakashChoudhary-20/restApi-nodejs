# restApi-nodejs

Basic CRUD operations implementation using ExpressJS and Mongoose.

## Implemented so far - 
- POST
- GET
  - GET ALL
  - GET BY ID

## Pending operations - 
- Update
- Delete

# Database schema
- name (String)
- ingredients (Array)
- diet (String)
- cook_time (Number)
- region (String)

# API Routes
- /api/v1/dishes**/**
- /api/v1/dishes**/get_all_dishes**
- /api/v1/dishes**/get_dish/:id**
